# WriCoRe Project Notes

## RAG Goal
The first RAG goal is to retrieve relevant knowledge-base chunks and return grounded answers with citations.

## Context Harness
The context harness controls what source material enters the model prompt.

## Constraint Harness
The constraint harness requires answers to cite chunk IDs and avoid hallucination.

## Evaluation Harness
The evaluation harness checks whether answers are grounded and supported by retrieved chunks.
